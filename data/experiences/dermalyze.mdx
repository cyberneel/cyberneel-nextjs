A deep-learning web app that classifies common skin conditions right in the browser.

=ReAdMoRe=

## Highlights
- Built a prototype deep-learning model classifying four skin conditions (acne, eczema, psoriasis, and moles) by fine-tuning a pre-trained ResNet on a custom dataset, achieving an 80% accuracy rate.
- Engineered a front-end with HTML Canvas and JavaScript to perform essential image pre-processing in the browser, ensuring the correct input format for the ONNX model.
- Prototyped the initial app in Streamlit, then migrated it to a static site on GitHub Pages for better performance and accessibility.
